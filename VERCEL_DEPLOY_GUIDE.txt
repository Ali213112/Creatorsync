===========================================
STEP 1: Sign Up / Log In to Vercel
===========================================

1. Go to: https://vercel.com
2. Click "Sign Up" (top right)
3. Choose "Continue with GitHub"
4. Authorize Vercel to access your GitHub account
5. You'll be redirected to Vercel dashboard

===========================================
STEP 2: Import Your Project
===========================================

1. In Vercel dashboard, click "Add New Project" button
2. You'll see a list of your GitHub repositories
3. Find "Creatorsync" in the list
4. Click "Import" next to Creatorsync

===========================================
STEP 3: Configure Project Settings
===========================================

You'll see a configuration page. Here's what to set:

PROJECT NAME:
- Keep default: "creatorsync" (or change if you want)

FRAMEWORK PRESET:
- Should auto-detect: "Next.js"
- If not, select "Next.js"

ROOT DIRECTORY:
- Leave as "." (default - means root folder)

BUILD COMMAND:
- Should be: "npm run build" (auto-filled)
- If empty, type: npm run build

OUTPUT DIRECTORY:
- Leave empty (Next.js handles this automatically)

INSTALL COMMAND:
- Should be: "npm install" (auto-filled)
- If empty, type: npm install

===========================================
STEP 4: Add Environment Variables
===========================================

IMPORTANT: Do this BEFORE clicking Deploy!

1. Click "Environment Variables" section (below project settings)
2. Click "Add" or the "+" button
3. Add these variables one by one:

   VARIABLE 1:
   Name: NEXT_PUBLIC_STORY_PROTOCOL_NETWORK
   Value: aeneid
   Environment: Production, Preview, Development (select all)
   Click "Save"

   VARIABLE 2 (Choose ONE - OpenAI OR Hugging Face):
   
   Option A - If you have OpenAI key:
   Name: OPENAI_API_KEY
   Value: (paste your OpenAI API key)
   Environment: Production, Preview, Development (select all)
   Click "Save"
   
   Option B - If you have Hugging Face key:
   Name: HUGGINGFACE_API_KEY
   Value: (paste your Hugging Face API key)
   Environment: Production, Preview, Development (select all)
   Click "Save"

   VARIABLE 3 (Optional - Story Protocol API key):
   Name: STORY_PROTOCOL_API_KEY
   Value: (your Story Protocol API key - if you have one)
   Environment: Production, Preview, Development (select all)
   Click "Save"

4. After adding all variables, you should see them listed

===========================================
STEP 5: Deploy!
===========================================

1. Scroll down and click the big blue "Deploy" button
2. Wait 2-3 minutes while Vercel:
   - Installs dependencies (npm install)
   - Builds your app (npm run build)
   - Deploys to production
3. You'll see a progress bar and logs
4. When it says "Ready" - you're done! ðŸŽ‰
