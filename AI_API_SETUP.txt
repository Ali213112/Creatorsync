===========================================
AI API KEY SETUP - HUGGING FACE
===========================================

✅ YES! Your code uses Hugging Face API key!

===========================================
HOW IT WORKS:
===========================================

1. **Tries Hugging Face FIRST** (free, no credit card needed!)
   - Uses model: mistralai/Mistral-7B-Instruct-v0.2
   - Free tier available
   - No credit card required

2. **Falls back to OpenAI** if Hugging Face fails
   - Uses gpt-3.5-turbo
   - Requires OpenAI API key
   - Costs money (but cheaper model)

===========================================
ENVIRONMENT VARIABLES:
===========================================

The code checks for these variables (in order):
1. HUGGINGFACE_API_KEY
2. NEXT_PUBLIC_HUGGINGFACE_API_KEY
3. OPENAI_API_KEY (fallback)
4. NEXT_PUBLIC_OPENAI_API_KEY (fallback)

===========================================
HOW TO SET UP:
===========================================

LOCAL DEVELOPMENT (.env.local):
--------------------------------
Create or edit `.env.local` file:

HUGGINGFACE_API_KEY=your_huggingface_key_here

OR

OPENAI_API_KEY=your_openai_key_here

VERCEL DEPLOYMENT:
------------------
1. Go to Vercel dashboard
2. Project Settings → Environment Variables
3. Add:
   - Name: HUGGINGFACE_API_KEY
   - Value: (your key)
   - Environment: Production, Preview, Development
4. Save and redeploy

===========================================
GET HUGGING FACE API KEY (FREE):
===========================================

1. Go to: https://huggingface.co/settings/tokens
2. Sign up or log in (FREE!)
3. Click "New token"
4. Name it (e.g., "creatorsync")
5. Select "Read" permission
6. Copy the token
7. Add to .env.local or Vercel
